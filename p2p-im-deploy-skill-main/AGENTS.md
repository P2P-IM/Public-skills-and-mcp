# Repository Instructions

This repository contains the agent-facing deployment skill for P2P IM node provisioning.

- Keep `SKILL.md` concise and focused on the orchestration workflow.
- Put provider-specific commands and verification details in `references/`.
- Do not duplicate the VPS installer from `p2p-im-deploy` here.
- Do not commit generated credentials, setup tokens, private keys, `.env` files, database files, or cloud account output containing secrets.
- Keep paid cloud/domain actions behind explicit user confirmation in the skill instructions.
